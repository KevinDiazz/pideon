# PideON 🍕

Aplicación web para la gestión integral de pedidos en establecimientos de restauración.

## 📁 Estructura del proyecto

- **frontend/**: Aplicación SPA desarrollada con React y Vite.
- **backend/**: API REST desarrollada con Node.js, Express y Prisma.
- **docs/**: Documentación del proyecto.

## 🚀 Tecnologías

- Frontend: React, Vite, TailwindCSS
- Backend: Node.js, Express, Prisma
- Base de datos: PostgreSQL
- Despliegue: Vercel y Railway
